# **App Name**: StyleSwap

## Core Features:

- Location-Based Availability: Users select their location to view available clothing rentals in their region.
- Secure Login with OTP: Secure user authentication via mobile number with OTP verification.
- Clothing Upload: Users can upload clothes with images, descriptions, sizes, prices and availability dates.
- Categorized Browsing: Browse rental clothing items with a lot of well-organized categories.
- Rental Item Details: Display clothing details such as size, condition, rental period and total cost.
- Automated garment categorization: An AI powered tool that automatically assigns garment categories such as 'ethnic wear', 'formal attire', 'casual wear' or other common and user generated tags by understanding the pictures and description text provided by the user.
- Terms and Conditions: Display all the terms and conditions in order to cover responsibilities and other relevant details

## Style Guidelines:

- Primary color: Rose-pink (#D87093), echoing the brand's image, symbolizing modern style and approachability.
- Background color: Light pink (#F8E8EE), complementing the primary hue for an elegant backdrop.
- Accent color: Lilac (#C8A2C8), enhancing specific sections and calls-to-action.
- Font: 'Belleza' sans-serif font for headings and 'Alegreya' serif for body text; presenting fashion-forward aesthetics in an approachable style.
- Elegant and minimalistic icons reflecting the different categories, such as party, office and other styles.
- Clean, user-friendly layout with intuitive navigation for ease of browsing.
- Subtle transitions and loading animations for a smooth, modern experience.
